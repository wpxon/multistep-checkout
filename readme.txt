A MultiStep Checkout plugin for WooCommerce.

== Description ==

**MultiStep Checkout** plugin splits the WooCommerce Checkout process into simpler steps for your online store.

== Features ==
 
* Fully responsive. 
* Easy to use.
* Easy Installation. 
* Translation ready  


== Installation ==

1. Go to Plugins > Add New.
1. Search for "MultiStep Checkout".
1. Click "Install Now".

OR

1. Download the zip file.
1. Upload the zip file via Plugins > Add New > Upload.

Activate the plugin. Look for "Ajax Contact Form" in the admin menu.
